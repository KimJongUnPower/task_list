Проект создан для использования в локальной сети.
При выгрузке на хостинг настоятельно рекомендую вынести папку core за www.
Проект имеет очевидные не достатки в виде чистого самописа на php (выполнено в угоду скорости разработки и специфике применения) со всеми вытекающими в виде:
1. Статически указанного протокола http.
2. Повторяющиеся куски кода.
3. Отсутствие человека понятных URL.
4. Статически указанные пути к core (недостаток при выгрузке на хостинг придётся все переназначать).
5. В некоторых местах сложная логика подключение других страниц.
6. Не использование какого-либо паттерна программирования. 

Не удобный UI.
Необходимо создать БД sqlite3 в core
